Update - This project was stopped amidst progress to focus on higher priority projects.

The purpose of this program is to provide a tool for law enforcement to conveniently record interview data and be able to refer to it at a later time.
It will have the ability to create, view, edit and delete interview cards.
The purpose of this project is to create a program from scratch and take it through the steps to completion of a usable tool with proper ui on any desktop or laptop computer.
